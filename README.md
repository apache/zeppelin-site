## Zeppelin project website
This readme will walk you through building the Zeppelin website

## Requirement
- Docker

## Develop website
```bash
$ ./run-dev.sh
```
- Open http://127.0.0.1:4000

### Adding a new page

```bash
$  rake page name="new-page.md"
```

## Build  website
```bash
$ ./build-dev.sh
```
