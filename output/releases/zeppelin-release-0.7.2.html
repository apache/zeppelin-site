Apache Zeppelin Release 0.7.2

The Apache Zeppelin community is pleased to announce the availability of the 0.7.2 release.

The community put significant effort into improving Apache Zeppelin since the last release. 25 contributors provided 50+ patches for improvements and bug fixes. More than 40+ issues have been resolved.

We encourage to download the latest release. Feedback through the mailing lists is very welcome.


You can visit issue tracker for full list of issues that are resolved.
